A port of the Neovim colorscheme, [thorn.nvim](https://github.com/jpwol/thorn.nvim), made by me.

Features dark and light rich greens and low contrast UI to provide a pleasant working experience.

## Credits

Original theme for Neovim by [jpwol](https://github.com/jpwol/thorn.nvim)


## Variants

- Thorn Dark Warm
- Thorn Dark Cold  
- Thorn Light Warm
- Thorn Light Cold

## Installation

Install via Zed's extension manager:
1. Open command palette (Cmd+Shift+P)
2. Run "zed: extensions"
3. Search for "Thorn"
4. Click Install

### Extras

Additional goodies can be found [here](https://github.com/jpwol/thorn.nvim/tree/main/extras)!

- Ghostty themes
- Kitty themes
- Alacritty themes
- Btop themes
